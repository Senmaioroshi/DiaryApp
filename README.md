# 🏥 HealthDiary: Медицинское приложение

Профессиональное мобильное решение для мониторинга здоровья, сочетающее в себе дневник приема лекарств, трекер симптомов и калькулятор физических показателей.

## ✨ Основные модули

### 💊 Дневник приема лекарств
* **Учет:** Добавление названия, дозировки и времени приема.
* **Уведомления:** Система напоминаний через `AlarmManager` / `WorkManager`, которая присылает push-уведомления в заданное время.
* **История:** Возможность отмечать принятые препараты и просматривать архив.

### 🧠 Дневник головной боли (Трекер симптомов)
* **Детальный лог:** Фиксация интенсивности боли (шкала 1-10), триггеров (стресс, погода, питание) и продолжительности приступа.
* **Визуализация:** Интерактивный график изменения состояния во времени (библиотека MPAndroidChart).

### ⚖️ Калькулятор ИМТ
* **Расчет:** Мгновенное вычисление индекса массы тела по росту и весу.
* **Интерпретация:** Автоматический вывод статуса (недостаток веса, норма, избыток) с цветовой индикацией.

## 🛠 Технологический стек

| Технология | Назначение |
|---|---|
| **Java / Android SDK** | Основной язык и среда разработки |
| **Room Database** | Локальное хранение данных (лекарства, симптомы, история) |
| **Material Design 3** | Современный и чистый UI/UX интерфейс |
| **AlarmManager** | Фоновые задачи и точные уведомления |
| **View Binding** | Безопасная работа с элементами интерфейса |


---

## 🚀 Как запустить
1. Склонируйте репозиторий.
2. Откройте проект в **Android Studio**.
3. Убедитесь, что в `build.gradle` подключены зависимости для Room и Material Design.
4. Запустите на эмуляторе или реальном устройстве (API 26+).
